# Taste
> taste的意思名为常鲜，品尝，体验。我赞成王阳明的知行合一：知是行的意识，行是知的功夫。只有不断的探索才能进步，因此写了这个项目。
>

| 技术 | 说明 | 官网 |

| SpringSecurity | 认证和授权框架 | https://spring.io/projects/spring-security |
| MyBatisPlus | ORM框架 | https://baomidou.com/ |
| Redis | 内存数据存储 | https://redis.io/ |
